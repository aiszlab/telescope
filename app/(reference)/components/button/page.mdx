# Button 按钮

按钮用于开始一个即时操作。

## 何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

## API

### Button

| 参数     | 说明               | 类型                                             | 默认值      |
| -------- | ------------------ | ------------------------------------------------ | ----------- |
| variant  | 样式               | `"filled" \| "outlined" \| "text"`               | `"filled"`  |
| color    | 主调色             | `"primary" \| "secondary" \| "tertiary"`         | `"primary"` |
| shape    | 形状               | `"rounded"`                                      | `"rounded"` |
| size     | 尺寸               | `"small" \| "medium"`                            | `"medium"`  |
| children | 显示内容           | `React.ReactNode`                                | `void 0`    |
| disabled | 是否禁用           | `boolean`                                        | `false`     |
| prefix   | 前置内容           | `React.ReactNode`                                | `void 0`    |
| suffix   | 后置内容           | `React.ReactNode`                                | `void 0`    |
| ripple   | 是否开启水波纹特效 | `boolean`                                        | `true`      |
| onClick  | 点击事件           | `(event: MouseEvent<HTMLButtonElement>) => void` | `void 0`    |

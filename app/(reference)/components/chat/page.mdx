## API

### Chat Props

| 参数      | 说明               | 类型                                  | 默认值   |
| --------- | ------------------ | ------------------------------------- | -------- |
| onMessage | 问答的消息处理函数 | [`MessageReceiver`](#MessageReceiver) | `void 0` |

### MessageReceiver

```ts
interface MessageReceiver {
  (
    send: string,
    receiver: {
      next: (content: string) => void
      complete: () => void
      error: (error: Error) => void
    }
  ): void

  /**
   * @description
   * override
   */
  override?: boolean
}
```

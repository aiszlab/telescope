# Countable 可计数组件

携带计数功能的按钮。

## 何时使用

常用于一些特定场景，比如发送验证码之后的倒计时。

## API

继承自[`Button`](https://aisz.dev/components/button)

| 参数     | 说明     | 类型     | 默认值 |
| -------- | -------- | -------- | ------ |
| count    | 计数大小 | `number` | `60`   |
| interval | 计时间隔 | `number` | `1000` |

# DatePicker 日期选择器

输入或选择日期的控件。

## 何时使用

当用户需要输入一个日期，可以点击标准输入框，弹出日期面板进行选择。

## API

| 参数     | 说明               | 类型                     | 默认值   |
| -------- | ------------------ | ------------------------ | -------- |
| value    | 选中时间           | `Dayjs`                  | `void 0` |
| onChange | 时间选择的回调事件 | `(value: Dayjs) => void` | `void 0` |

# DateRangePicker 日期范围选择器

输入或选择日期范围的控件。

## 何时使用

当用户需要输入一个日期范围，可以点击标准输入框，弹出日期面板进行选择。

## API

| 参数     | 说明               | 类型                                       | 默认值   |
| -------- | ------------------ | ------------------------------------------ | -------- |
| value    | 选中的时间范围     | `[Dayjs \| undefined, Dayjs \| undefined]` | `void 0` |
| onChange | 选择范围的回调事件 | `(value: [Dayjs, Dayjs]) => void`          | `void 0` |

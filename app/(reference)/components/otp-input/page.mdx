## API

| 参数     | 说明       | 类型                      | 默认值   |
| -------- | ---------- | ------------------------- | -------- |
| length   | 输入位数   | `number`                  | `6`      |
| value    | 输入内容   | `string`                  | `void 0` |
| onChange | 变更的回调 | `(value: string) => void` | `void 0` |

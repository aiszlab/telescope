## 快速开始

```bash
pnpm i musae
```

All components are available in this one package for ease of dependency management.

## 如何使用

1. 在样式文件中全局引入组件库样式

```css
@import 'musae';
```

2. 在组件代码中使用组件

```tsx
import { Button } from 'musae'

const App = () => {
  return <Button>使用 musae</Button>
}

export default App
```

## 内置 Icons

```tsx
import { Button } from 'musae'
import { Github } from 'musae/icons'

const App = () => {
  return (
    <Button shape='circular' variant='text'>
      <Github />
    </Button>
  )
}

export default App
```

## 0 配置支持按需加载

在`musae`组件包编译阶段，使用`rollup`创建`ESModule`产物，天然支持`treeshaking`
如果开发者有兴趣，可以查阅[Github](https://github.com/aiszlab/musae)查看根目录下的`rollup.config.js`配置文件

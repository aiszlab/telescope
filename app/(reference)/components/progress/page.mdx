## API

### Progress Props

| 参数    | 说明       | 类型                     | 默认值     |
| ------- | ---------- | ------------------------ | ---------- |
| variant | 进度条样式 | `"linear" \| "circular"` | `"linear"` |
| value   | 进度条值   | `number`                 | `0`        |

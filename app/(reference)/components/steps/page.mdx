# Steps 步骤条

## 何时使用

步骤条，一般用于完成一个任务需要分几个步骤，在每个步骤完成之前，用户必须完成前一个步骤。

## API

### Steps

| 参数     | 说明                                                           | 类型                         | 默认值         |
| -------- | -------------------------------------------------------------- | ---------------------------- | -------------- |
| items    | 步骤项目列表                                                   | `StepItem[]`                 | -              |
| value    | 当前所在步骤                                                   | `number`                     | `0`            |
| onChange | 在步骤条中点击步骤的回调事件，不设置的场景下认为步骤条不可点击 | `(value: number) => void`    | `void 0`       |
| type     | 类型                                                           | `"horizontal" \| "vertical"` | `"horizontal"` |
| size     | `item.leading` 的大小，如果指定大小，按照指定的大小进行渲染    | `number`                     | `void 0`       |

### StepItem

| 参数        | 说明       | 类型              | 默认值   |
| ----------- | ---------- | ----------------- | -------- |
| leading     | 步骤条图标 | `React.ReactNode` | `void 0` |
| title       | 步骤条标题 | `React.ReactNode` | -        |
| description | 步骤条描述 | `React.ReactNode` | `void 0` |

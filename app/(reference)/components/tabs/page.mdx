# Tabs 标签页

选项卡切换组件。

## 何时使用

提供平级的区域将大块内容进行收纳和展现，保持界面整洁。

- 卡片式的页签，提供可关闭的样式，常用于容器顶部。
- 既可用于容器顶部，也可用于容器内部，是最通用的 Tabs。

## API

### Tabs

| 参数             | 说明                                | 类型        | 默认值   |
| ---------------- | ----------------------------------- | ----------- | -------- |
| items            | 对应的明细项，用于切换              | `TabItem[]` | `[]`     |
| activeKey        | 受控的选中值                        | `React.Key` | `void 0` |
| defaultActiveKey | 默认的选中值，优先级低于`activeKey` | `React.Key` | `void 0` |
| forceRender      | 在初始化渲染时强制渲染所有面板      | `boolean`   | `false`  |
| destroyable      | 面板在隐藏时是否直接卸载            | `boolean`   | `false`  |

### TabItem

| 参数     | 说明                   | 类型              | 默认值   |
| -------- | ---------------------- | ----------------- | -------- |
| key      | 当前子项唯一键         | `React.Key`       | -        |
| label    | 当前子项展示内容       | `string`          | -        |
| children | 当前子项渲染的面板内容 | `React.ReactNode` | `void 0` |

## API

### Tag Props

| 参数     | 说明             | 类型                             | 默认值     |
| -------- | ---------------- | -------------------------------- | ---------- |
| children | 标签显示内容     | `React.ReactNode`                | `void 0`   |
| size     | 图片描述         | `"small" \| "medium" \| "large"` | `"medium"` |
| closable | 是否具备关闭功能 | `boolean`                        | `false`    |
| onClose  | 关闭的回调事件   | `() => void`                     | `void 0`   |
| leading  | 显示前置组件     | `React.ReactNode`                | `void 0`   |

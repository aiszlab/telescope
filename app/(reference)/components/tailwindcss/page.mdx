## 定制 musae 样式

在`musae`中，本身已经提供了一套完整的`CSS`样式，包括按钮、表单、表格等组件。
开发者本身是不需要关心任何`CSS`，也不需要手动引入任何 CSS 文件。
但是样式定制是永远避免不了，这里我们以集成`tailwindcss`为例。

## 配置 tailwindcss

在使用`tailwindcss`时，我们根据`tailwindcss setup`，通常会创建一个根目录`css`文件，以下我们统称`globals.css`，在其中引入`tailwindcss`

```css
@import 'tailwindcss';
```

如果直接在`globals.css`引入`musae`样式，那么会造成预期之外的破坏，部分原子化样式无法覆盖组件原有样式。

## 调整样式优先级

修改`globals.css`

```css
@import 'tailwindcss';

@layer components {
  @import 'musae';
}
```

完成以上修改就可以完美集成`tailwindcss`与`musae`了 🎉🎉🎉。

## API

### Textarea Props

| 参数     | 说明                   | 类型                      | 默认值   |
| -------- | ---------------------- | ------------------------- | -------- |
| value    | 文本内容               | `string`                  | `void 0` |
| invalid  | 不符合规则             | `boolean`                 | `void 0` |
| onChange | 内容发生改变的回调函数 | `(value: string) => void` | `void 0` |

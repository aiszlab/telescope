# TimePicker 时间选择器

输入或选择时间的控件。

## 何时使用

当用户需要输入一个时间，可以点击标准输入框，弹出时间面板进行选择。

## API

| 参数     | 说明               | 类型                     | 默认值   |
| -------- | ------------------ | ------------------------ | -------- |
| value    | 选中的时间         | `Dayjs`                  | `void 0` |
| onChange | 选择范围的回调事件 | `(value: Dayjs) => void` | `void 0` |

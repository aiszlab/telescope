## API

### Timeline Props

| 参数  | 说明                                                    | 类型                               | 默认值    |
| ----- | ------------------------------------------------------- | ---------------------------------- | --------- |
| items | 时间线项目列表                                          | `TimelineItem[]`                   | -         |
| mode  | 时间线布局模式                                          | `"left" \| "right" \| "alternate"` | `"right"` |
| size  | `item.dot` 的大小，如果指定大小，按照指定的大小进行渲染 | `number`                           | `void 0`  |

### TimelineItem

| 参数        | 说明           | 类型              | 默认值   |
| ----------- | -------------- | ----------------- | -------- |
| label       | 节点展示标签   | `React.ReactNode` | `void 0` |
| description | 节点展示描述   | `React.ReactNode` | -        |
| dot         | 自定义节点标识 | `React.ReactNode` | `void 0` |

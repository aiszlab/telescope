## API

| 参数       | 说明           | 类型                                                    | 默认值  |
| ---------- | -------------- | ------------------------------------------------------- | ------- |
| columns    | 列数           | `number`                                                | `4`     |
| gutter     | 项目之间的间距 | `number` \| `[number, number]`                          | `8`     |
| children   | 项目列表       | `Exclude<React.ReactNode, Iterable<React.ReactNode>>[]` | `[]`    |
| sequential | 有序流         | `boolean`                                               | `false` |

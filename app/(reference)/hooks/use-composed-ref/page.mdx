## API

### Parameters

| 参数 | 说明                 | 类型                               |
| ---- | -------------------- | ---------------------------------- |
| refs | 需要被组合的挂载对象 | `Voidable<Refable<Nullable<T>>>[]` |

#### Refable

```ts
type Refable<T> = React.RefCallback<T> | React.MutableRefObject<T> | string
```

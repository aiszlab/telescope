## API

### Parameters

| 参数      | 说明     | 类型                   | 默认值 |
| --------- | -------- | ---------------------- | ------ |
| debouncer | 执行函数 | `T \| Debouncer<T, R>` | -      |
| wait      | 等待时间 | `number`               | 1000   |

#### Debouncer

```ts
type Debouncer<T extends Callable, R extends Array<unknown> = Parameters<T>> = {
  callback: (...args: R) => ReturnType<T>
  pipeable: Nullable<(...args: Parameters<T>) => R | Promise<R>>
}
```

### Return

| 参数  | 说明             | 类型                               |
| ----- | ---------------- | ---------------------------------- |
| next  | 防抖函数         | `(...args: Parameters<T>) => void` |
| flush | 取消防抖限制     | `() => void`                       |
| abort | 中断执行中的函数 | `() => void`                       |

# useHover

处理元素的指针悬停交互。规范化浏览器和平台之间的行为，并忽略触摸设备上模拟的鼠标事件。

## API

### Parameters

| 参数    | 说明                   | 类型                      | 默认值 |
| ------- | ---------------------- | ------------------------- | ------ |
| onEnter | 鼠标移入当前元素的事件 | `(e: HoverEvent) => void` | -      |
| onLeave | 鼠标离开当前元素的事件 | `(e: HoverEvent) => void` | -      |

### Return

#### UsedHover

| 参数       | 说明                  | 类型         |
| ---------- | --------------------- | ------------ |
| isHovered  | 当前元素是否被`hover` | `boolean`    |
| hoverProps | 元素`hover`相关的属性 | `HoverProps` |

#### HoverProps

| 参数           | 说明                   | 类型         |
| -------------- | ---------------------- | ------------ |
| onPointerEnter | 鼠标移入当前元素的事件 | `() => void` |
| onPointerLeave | 鼠标移出当前元素的事件 | `() => void` |

## API

### Parameters

| 参数   | 说明                         | 类型     |
| ------ | ---------------------------- | -------- |
| signal | 特殊标识，保证当前组件内唯一 | `string` |

### Return

| 参数     | 说明                            | 类型           |
| -------- | ------------------------------- | -------------- |
| id       | 当前组件`id`，参考`React.useId` | `string`       |
| uniqueId | 创建组件范围内的唯一`id`        | `() => string` |

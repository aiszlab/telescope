## clsx 合并类名

## 使用场景

在组件开发过程中，通常`className`可能来自很多地方：

1. `props.className`
2. `CSS Module`
3. 根据判断是否展现

`clsx`能够快速兼容和处理上述场景

## 代码演示

```ts
import { clsx } from '@aiszlab/relax'

const CLASS_NAMES = [
  'test-1',
  false && 'test-2',
  true && 'test-3',
  ['test-array-item-1', 'test-array-item-2'],
  {
    'test-object-item-1': true,
    'test-object-item-2': false
  }
]

clsx(CLASS_NAMES) // => "test-1 test-3 test-array-item-1 test-array-item-2 test-object-item-1"
```

## API

### Avatar

| 参数  | 说明                     | 类型     | 默认值     |
| ----- | ------------------------ | -------- | ---------- |
| src   | 资源地址                 | `string` | -          |
| alt   | 图像无法显示时的替代文本 | `string` | -          |
| size  | 设置头像的大小           | `Size`   | `medium`   |
| shape | 指定头像的形状           | `Shape`  | `circular` |

### Size

```ts
type Size = 'small' | 'medium' | 'large'
```

### Shape

```ts
type Shape = 'circular' | 'squared'
```

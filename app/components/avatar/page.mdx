## API

### Avatar

| 参数     | 说明                     | 类型                             | 默认值       |
| -------- | ------------------------ | -------------------------------- | ------------ |
| src      | 资源地址                 | `string`                         | `void 0`     |
| alt      | 图像无法显示时的替代文本 | `string`                         | `void 0`     |
| size     | 设置头像的大小           | `"small" \| "medium" \| "large"` | `"medium"`   |
| shape    | 指定头像的形状           | `"circular" \| "squared"`        | `"circular"` |
| fallback | 无法显示图片时的兜底内容 | `React.ReactNode`                | `void 0`     |

### Group

| 参数     | 说明           | 类型                             | 默认值       |
| -------- | -------------- | -------------------------------- | ------------ |
| children | 子组件         | `React.ReactNode`                | -            |
| max      | 最多显示个数   | `number`                         | `3`          |
| size     | 设置头像的大小 | `"small" \| "medium" \| "large"` | `"medium"`   |
| shape    | 指定头像的形状 | `"circular" \| "squared"`        | `"circular"` |

## API

### Badge

| 参数      | 说明                       | 类型                                                           | 默认值        |
| --------- | -------------------------- | -------------------------------------------------------------- | ------------- |
| children  | 徽标挂在的元素             | `React.ReactNode`                                              | -             |
| content   | 徽标的内容，为空时显示为点 | `React.ReactNode`                                              | `void 0`      |
| invisible | 徽标是否隐藏               | `boolean`                                                      | `false`       |
| placement | 徽标定位                   | `"top-right" \| "top-left" \| "bottom-right" \| "bottom-left"` | `"top-right"` |

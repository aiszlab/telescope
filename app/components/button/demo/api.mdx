通用属性参考：[通用属性](/docs/react/common-props)

通过设置 Button 的属性来产生不同的按钮样式，推荐顺序为：`type` -> `shape` -> `size` -> `loading` -> `disabled`。

按钮的属性说明如下：

| 属性  | 说明                           | 类型    | 默认值 | 版本 |
| ----- | ------------------------------ | ------- | ------ | ---- |
| block | 将按钮宽度调整为其父宽度的选项 | boolean | false  |      |

支持原生 button 的其他所有属性。

## API

| 参数      | 说明                   | 类型                        | 默认值 |
| --------- | ---------------------- | --------------------------- | ------ |
| value     | 当前时间点或者时间区间 | `Dayjs` \| `[Dayjs, Dayjs]` | -      |
| onClick   | 点击日期之后的回调事件 | (clickAt: `Dayjs`) => void  | -      |
| focusedAt | 顶部聚焦的年月         | `Dayjs`                     | -      |

## API

### Clock

| 参数     | 说明             | 类型                                        | 默认值   |
| -------- | ---------------- | ------------------------------------------- | -------- |
| value    | 时间值           | `[number, number, number]`                  | `void 0` |
| onChange | 修改时间后的回调 | `(value: [number, number, number]) => void` | `void 0` |

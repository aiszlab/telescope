## API

### Clock Props

| 参数             | 说明               | 类型                           | 默认值   |
| ---------------- | ------------------ | ------------------------------ | -------- |
| defaultActiveKey | 默认打开的面板     | `React.Key \| React.Key[]`     | `void 0` |
| activeKey        | 受控打开的面板     | `React.Key \| React.Key[]`     | `void 0` |
| items            | 面板列表           | `CollapseItem[]`               | -        |
| accordion        | 手风琴模式         | `boolean`                      | `false`  |
| onChange         | 面板展开的回调函数 | `(value: React.Key[]) => void` | `void 0` |

### CollapseItem

| 参数     | 说明             | 类型              | 默认值 |
| -------- | ---------------- | ----------------- | ------ |
| key      | 面板对应的唯一值 | `React.Key`       | -      |
| label    | 面板的标题       | `string`          | -      |
| children | 面板的主体内容   | `React.ReactNode` | -      |

# Dialog 对话框

模态对话框。

## 何时使用

需要用户处理事务，又不希望跳转页面以致打断工作流程时，可以使用`Dialog`在当前页面正中打开一个浮层，承载相应的操作。
另外当需要一个简洁的确认框询问用户时，可以使用`Dialog.confirm()`等语法糖方法。

## API

| 参数      | 说明                     | 类型           | 默认值  |
| --------- | ------------------------ | -------------- | ------- |
| title     | 对话框的标题             | `ReactNode`    | -       |
| children  | 对话框的内容             | `ReactNode`    | -       |
| open      | 控制对话框是否打开       | `boolean`      | `false` |
| footer    | 对话框底部的定制内容     | `ReactNode`    | -       |
| onConfirm | 对话框点击确认的回调函数 | `VoidFunction` | -       |
| onClose   | 对话框点击关闭的回调函数 | `VoidFunction` | -       |

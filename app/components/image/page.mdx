## API

### Image Props

| 参数   | 说明         | 类型                 | 默认值   |
| ------ | ------------ | -------------------- | -------- |
| src    | 图片资源地址 | `string`             | -        |
| alt    | 图片描述     | `string`             | `void 0` |
| width  | 图片宽度     | `number` \| `string` | `void 0` |
| height | 图片高度     | `number` \| `string` | `void 0` |

### Group Props

| 参数     | 说明         | 类型              | 默认值 |
| -------- | ------------ | ----------------- | ------ |
| items    | 图片资源列表 | `string[]`        | -      |
| children | 子元素       | `React.ReactNode` | -      |

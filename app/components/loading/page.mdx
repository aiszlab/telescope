# Loading 加载中

用于页面和区块的加载中状态。

## 何时使用

页面和区块的加载中状态。

## 代码演示

页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

## API

### Loading Props

| 参数     | 说明           | 类型                             | 默认值     |
| -------- | -------------- | -------------------------------- | ---------- |
| size     | 尺寸           | `"small" \| "medium" \| "large"` | `"medium"` |
| overlay  | 是否显示遮盖层 | `boolean`                        | `true`     |
| children | 子组件         | `React.ReactNode`                | `void 0`   |

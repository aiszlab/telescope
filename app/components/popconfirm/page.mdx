## API

### Popconfirm Props

| 参数        | 说明               | 类型                                   | 默认值     |
| ----------- | ------------------ | -------------------------------------- | ---------- |
| title       | 标题               | `React.ReactNode`                      | `void 0`   |
| description | 内容               | `React.ReactNode`                      | `void 0`   |
| children    | 需要展示气泡的组件 | `React.ReactElement<P>`                | -          |
| placement   | 气泡展示位置       | `import("@floating-ui/dom").Placement` | `"bottom"` |
| onConfirm   | 确认回调函数       | `() => void`                           | `void 0`   |
| onCancel    | 取消回调函数       | `() => void`                           | `void 0`   |

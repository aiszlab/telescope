# Radio 单选框

单选框。

## 何时使用

- 用于在多个备选项中选中单个状态。
- 和`Select`的区别是`Radio`所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

## API

### Radio

| 参数     | 说明           | 类型      | 默认值 |
| -------- | -------------- | --------- | ------ |
| value    | 单选框对应的值 | `Key`     | -      |
| checked  | 选中单选框     | `boolean` | -      |
| disabled | 禁用单选框     | `boolean` | -      |

### Group

| 参数     | 说明       | 类型      | 默认值 |
| -------- | ---------- | --------- | ------ |
| value    | 选中的值   | `Key`     | -      |
| disabled | 禁用单选框 | `boolean` | -      |

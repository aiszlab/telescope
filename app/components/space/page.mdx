## API

| 参数     | 说明             | 类型                           | 默认值   |
| -------- | ---------------- | ------------------------------ | -------- |
| gutter   | 子元素之间的边距 | `number` \| `[number, number]` | `4`      |
| children | 子元素           | `React.ReactNode`              | `void 0` |

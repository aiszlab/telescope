# Table 表格

展示行列数据。

## 何时使用

- 当有大量结构化的数据需要展现时；
- 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。

## API

### Table Props

| 参数       | 说明           | 类型                                               | 默认值   |
| ---------- | -------------- | -------------------------------------------------- | -------- |
| dataSource | 数据列表       | `T[]`                                              | `[]`     |
| columns    | 表格列定义回调 | `(helper: ColumnHelper<T>) => ColumnDef<T, any>[]` | `void 0` |
| bordered   | 是否展示边框   | `boolean`                                          | `false`  |

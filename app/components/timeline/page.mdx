## API

### Timeline

| 参数     | 说明                                                           | 类型                         | 默认值         |
| -------- | -------------------------------------------------------------- | ---------------------------- | -------------- |
| items    | 步骤项目列表                                                   | `StepItem[]`                 | -              |
| value    | 当前所在步骤                                                   | `number`                     | `0`            |
| onChange | 在步骤条中点击步骤的回调事件，不设置的场景下认为步骤条不可点击 | `(value: number) => void`    | `void 0`       |
| type     | 类型                                                           | `"horizontal" \| "vertical"` | `"horizontal"` |

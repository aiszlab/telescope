## API

### Tour

| 参数             | 说明           | 类型                         | 默认值   |
| ---------------- | -------------- | ---------------------------- | -------- |
| open             | 是否显示       | `boolean`                    | `false`  |
| steps            | 步骤列表       | `TourStep[]`                 | -        |
| onClose          | 结束引导的回调 | `() => void`                 | `void 0` |
| overlay          | 是否显示遮盖层 | `boolean`                    | `true`   |
| spotlightPadding | 聚光灯的内边距 | `number \| [number, number]` | `8`      |

### TourStep

```ts
export type TourStep = {
  /**
   * @description
   * title
   */
  title: ReactNode

  /**
   * @description
   * description
   */
  description?: ReactNode

  /**
   * @description
   * target
   */
  target: Nullable<Element> | (() => Nullable<Element>)
}
```

## API

| 参数     | 说明             | 类型                      | 默认值   |
| -------- | ---------------- | ------------------------- | -------- |
| count    | 总分             | `number`                  | `5`      |
| value    | 受控值           | `number`                  | `void 0` |
| onChange | 值变更的回调事件 | `(value: number) => void` | `void 0` |
| halfable | 是否支持半选     | `boolean`                 | `true`   |
| disabled | 是否禁用         | `boolean`                 | `false`  |

# Tree 树形

多层次的结构列表。

## 何时使用

文件夹、组织架构、生物分类、国家地区等等，世间万物的大多数结构都是树形结构。使用`树控件`可以完整展现其中的层级关系，并具有展开收起选择等交互功能。

## API

### Tree Props

| 参数         | 说明                       | 类型                          | 默认值   |
| ------------ | -------------------------- | ----------------------------- | -------- |
| nodes        | 节点列表                   | `TreeNode[]`                  | `[]`     |
| expandedKeys | 展开的节点`key`            | `React.Key[]`                 | `void 0` |
| onExpand     | 展开的回调事件             | `(keys: React.Key[]) => void` | `void 0` |
| selectable   | 树节点是否支持高亮选中功能 | `boolean`                     | `true`   |
| selectedKeys | 被选中高亮的`Key`列表      | `React.Key[]`                 | `void 0` |

### TreeNode

| 参数     | 说明         | 类型              | 默认值   |
| -------- | ------------ | ----------------- | -------- |
| key      | 节点唯一键   | `React.Key`       | -        |
| title    | 节点显示文案 | `React.ReactNode` | -        |
| children | 子节点       | `TreeNode[]`      | `void 0` |

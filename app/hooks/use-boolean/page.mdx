## API

| 参数         | 说明   | 类型                       | 默认值 |
| ------------ | ------ | -------------------------- | ------ |
| initialState | 初始值 | `boolean \| () => boolean` | -      |

## Return

| 参数       | 说明         | 类型                                |
| ---------- | ------------ | ----------------------------------- |
| isOn       | 真假值       | `number \| () => number`            |
| turnOn     | 设置为真     | `() => void`                        |
| turnOff    | 设置为假     | `() => void`                        |
| toggle     | 设置为相反值 | `() => void`                        |
| setBoolean | 设置函数     | `Dispatch<SetStateAction<boolean>>` |

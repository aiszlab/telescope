## API

| 参数     | 说明     | 类型     | 默认值 |
| -------- | -------- | -------- | ------ |
| callback | 执行函数 | `T`      | -      |
| wait     | 等待时间 | `number` | 1000   |

## Return

## Return

| 参数     | 说明             | 类型         |
| -------- | ---------------- | ------------ |
| next     | 防抖函数         | `T`          |
| complete | 取消防抖限制     | `() => void` |
| cancel   | 中断执行中的函数 | `() => void` |

| 参数        | 说明     | 类型                                            | 默认值     |
| ----------- | -------- | ----------------------------------------------- | ---------- |
| label       | 提示文案 | string                                          | -          |
| placeholder | 兜底文案 | string                                          | -          |
| variant     | 兜底文案 | `outlined` \| `filled` \| `standard`            | `outlined` |
| prefix      | 前缀组件 | ReactNode                                       | -          |
| suffix      | 后缀组件 | ReactNode                                       | -          |
| type        | 类型     | `text` \| `password`                            | `text`     |
| value       | 值       | string                                          | -          |
| className   | 样式名   | string                                          | -          |
| onChange    | 变更事件 | `(value: string) => void`                       | -          |
| onFocus     | 样式名   | `(event: FocusEvent<HTMLInputElement>) => void` | -          |
| onBlur      | 样式名   | `(event: FocusEvent<HTMLInputElement>) => void` | -          |

### Radio

| 参数     | 说明                                 | 类型             | 默认值 |
| -------- | ------------------------------------ | ---------------- | ------ |
| checked  | 指定当前是否被选中                   | boolean          | -      |
| value    | 单选框对应的值，比较后判断是否被选中 | string \| number | -      |
| children | 子组件                               | ReactNode        | -      |
| disabled | 是否禁用                             | boolean          | -      |

### Radio.Group

| 参数     | 说明                | 类型             | 默认值 |
| -------- | ------------------- | ---------------- | ------ |
| value    | 当前 radio 组选中值 | string \| number | -      |
| children | 子组件              | ReactNode        | -      |
| disabled | 是否禁用            | boolean          | -      |
